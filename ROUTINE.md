# 秘書Routine 運用仕様

村長（Daisuke）はAuDHDのため、一度に多くの情報を渡さない。
このRoutineは「今やるべきこと」だけを、理由つきで短く伝えることに専念する。

## 実行タイミング

1日3回：朝・昼・晩（目安：8:00 / 13:00 / 20:00）

## 毎回やること

1. このリポジトリ内の全メモ（引き継ぎメモ・壁打ちメモ・各事業メモ）を読む
2. 日付・締切・健康・資金状況など「今この瞬間の状況」を踏まえて、
   村長が今やるべきことを **3つだけ** 選ぶ
3. 各項目に一言だけ理由をつける（長文の背景説明はしない）
4. Slack DM（宛先：U0BCFLPEW1Y）に送る

## 出力フォーマット【厳守】

- 3項目だけ。4つ以上にしない
- 各項目は「やること → 一言理由」の2行以内
- 前置き・まとめ・絵文字過多な装飾は避ける（最低限の見出しのみ可）
- 同じ内容を繰り返さない。前回送った内容と同じなら、進捗確認に言い換える

例：

```
🌙 今やること（晩）

1️⃣ やること
→ 一言理由

2️⃣ やること
→ 一言理由

3️⃣ やること
→ 一言理由
```

## 優先順位の判断基準（上から優先）

1. 締切・期限が迫っているもの（資金・通院・支払い等）
2. 健康に関わるもの
3. 「本体」であるカタカムラを守るための燃料調達（クラウドワークス・道の駅アプリ等のビジネス）
4. 習慣維持系（X投稿など）— ただし他に緊急事項がなければ優先度を上げる

参照：`引き継ぎメモ追記_深層心理_20260622.md` の結論
「ビジネスは燃料調達、本体はカタカムラ」を判断の前提とする。
