# hisyo

村長（Daisuke）の秘書Routine用データ置き場。

## ここに置くもの

- 引き継ぎメモ（人格・ルール・進行中プロジェクトの全体像）
- 各事業のメモ
  - カタカムラ
  - iPad日和
  - 杜花コーヒー
  - ミコトからの手紙

## 使い方

Claude Code Routinesが、ここに置かれたメモを読んで
「今やるべきこと・優先順位」を朝・昼・晩に判断し、Slackに通知する。

- 通知先Slackチャンネル：`#村長-今やること`（チャンネルID: `C0BDYK45UMS`）
- 通知形式：見出し＋「今やること」5項目のみ（AuDHD対応のため情報量を絞る）
- 朝・昼・晩のスケジュール実行は、Claude Code on the webのTrigger（環境のスケジュール設定）で設定すること。
  セッション内のCronCreateはコンテナ終了で消えるため、恒久的な3回/日実行には使えない。

## 更新ルール

各メモを更新したら、このリポジトリにも反映すること。
（自動同期ではないため、手動アップロードが必要）
