# hisyo

村長（Daisuke）の秘書Routine用データ置き場。

## ここに置くもの

- 引き継ぎメモ（人格・ルール・進行中プロジェクトの全体像）
- 各事業のメモ
  - カタカムラ
  - iPad日和
  - 杜花コーヒー
  - ミコトからの手紙

## 使い方

Claude Code Routinesが、ここに置かれたメモを読んで
「今やるべきこと・優先順位」を朝・昼・晩に判断し、Slackに通知する。

## 更新ルール

- 各メモを更新したら、このリポジトリにも必ず反映すること。
  （自動同期ではないため、手動アップロードが必要）
- 既存ファイルを上書きするだけで済ませてはならない。修正点は必ず残すこと。
  - 「何を」「なぜ」直したのか（変更前→変更後）を明記する
  - 追記は「◯◯追記_YYYYMMDD.md」の形式で新規ファイルとして追加する
  - 追記が増えてきたら「統合版」ファイルにまとめ直し、経緯が追えるようにする
