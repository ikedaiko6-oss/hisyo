## 追記内容【2026.06.03】

### 道の駅アプリ（既に完成済み）

- 技術スタック：Next.js + React + TypeScript + Tailwind CSS + Supabase + Vercel + OpenStreetMap
- フェーズ1：無料でデータを育てる
- フェーズ2：プレミアム機能追加で課金

### 収益モデル：B2Bモデル【盲点から生まれた核心】

- ユーザーから取るのではなく、**道の駅側から課金**（月額3,000〜5,000円/店舗）
- 価値：入荷情報・売り切れプッシュ通知を道の駅が発信→客の無駄足をなくす
- 道の駅のメリット：売り切れ前に来てもらえる・客単価向上・廃棄減少

### 実装方針

- Stripe + Supabase Webhook で課金管理
- 作り直し不要。現行スタックで全部できる
- CodexのSites機能・Businessプランは不要。Vercelのまま
